# vscode-qrcode
## 描述
方便地生成和显示二维码,支持将图片保存为`png`或`svg`
## 命令
* `QRCode:预览二维码`:支持文本编辑器右键菜单

## 注意
vscode刚打开时就打开二维码界面容易出现画面崩坏,可以先关闭界面等vscode加载完所有插件后再运行命令
